## 문제 풀이
### 아이디어
- 스택에는 ‘높이가 증가하는 막대의 인덱스’만 저장한다.  
  막대가 감소하는 순간, 그 전까지 쌓아온 막대들 중 자기보다 큰 막대들은 이제 더 이상 확장될 수 없기 때문.
- 스택에서 꺼내면서 '이 막대를 높이로 쓰는 최대 너비' 를 계산한다.

### 로직
1) 스택에는 **오름차순 높이의 인덱스**만 쌓는다.
2) 더 낮은 막대가 등장하면,  
   pop 하면서 **이 막대를 높이로 하는 직사각형의 최대 너비** 를 계산한다.  
  - 스택 top = pop한 막대보다 **왼쪽에서 처음으로 작은 막대**
  - pop된 막대의 최대 확장 범위: `width = 현재 인덱스 - stack.top `

3) 모든 막대를 체크한 후, 스택에 남아 있는 막대들도 동일하게 pop 하면서 계산
   → `h[n + 1] = 0`이기 때문에 스택에 남아있는 막대들도 계산할 수 있다.
